# Retail Market Survey Analysis

## Project Overview

This project is an Excel-based retail market research and data analysis project focused on the biscuit category.

The project involved personally collecting primary data from retail shops in New Market, organising the collected information, preparing the dataset, performing PivotTable-based analysis, and developing an Excel dashboard to generate meaningful market insights.

---

## Project Objective

The objective of this project was to understand product and brand presence within the selected retail market and identify meaningful patterns related to product availability, pricing, and market presence.
